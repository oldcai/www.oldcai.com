    FastMD

    FastMD is a native, preview-first Markdown editor for macOS.

    • System requirements: macOS 14 Sonoma or later; Apple Silicon
    • Supported files: .md, .markdown, and .mdown
    • Account required: No
    • Data handling: Local by default; see the Privacy Policy

    Quick start

    1. Open FastMD and choose Open Markdown File…, or open a Markdown file from Finder.
    2. Read the rendered document, then click a text block to edit it in place.
    3. Use View → Source Mode to edit the complete Markdown source.
    4. Select a Markdown file in Finder and press Space to use the bundled Quick Look preview.

    Frequently asked questions

    Why does FastMD ask for folder access?
    Folder access lets FastMD display relative local images, show Markdown files in the same folder, follow atomic saves made by other editors, and remember access between launches. You can choose Not Now; the open document remains usable, but those folder-dependent features may be limited.

    Does FastMD upload my documents?
    Not in the background. Documents stay on your Mac unless you explicitly choose Publish Preview via Cloudflare Drop…. Before any transfer, FastMD explains that it will send an HTML snapshot and referenced local resources to Cloudflare and asks you to continue. The resulting link is temporarily public.

    Why does a Markdown feature appear as source text?
    FastMD intentionally implements a focused, safe Markdown subset. Unsupported or ambiguous constructs remain editable as source rather than being discarded or rewritten.

    How do I report a problem?
    Include your macOS version, the smallest Markdown example that reproduces the issue, what you expected, and what happened.

    FastMD — 支持

    FastMD 是一款原生、预览优先的 macOS Markdown 编辑器。

    • 系统要求： macOS 14 Sonoma 或更高版本；Apple Silicon
    • 支持文件： .md、.markdown、.mdown
    • 是否需要账户： 不需要
    • 数据处理： 默认仅在本机；详见隐私政策

    快速上手

    1. 打开 FastMD 并选择 Open Markdown File…，也可以从访达直接打开 Markdown 文件。
    2. 阅读排版后的文档，点击文字块即可原位编辑。
    3. 通过 View → Source Mode 编辑完整 Markdown 源码。
    4. 在访达中选中 Markdown 文件并按空格键，可使用随附的“快速查看”扩展。

    常见问题

    为什么 FastMD 会请求文件夹访问权限？
    该权限用于显示相对路径本地图片、列出同文件夹 Markdown 文件、跟随其他编辑器的原子保存，并在下次启动时继续访问。你可以选择 Not Now；当前文档仍可使用，但依赖文件夹的功能可能受限。

    FastMD 会上传文档吗？
    不会在后台上传。除非你主动选择 Publish Preview via Cloudflare Drop…，文档始终留在 Mac 上。传输前，FastMD 会明确说明将 HTML 快照和引用的本地资源发送给 Cloudflare，并要求你确认继续。生成的链接会暂时公开。

    如何反馈问题？
    请附上 macOS 版本、可复现问题的最小 Markdown 示例、预期结果和实际结果。
