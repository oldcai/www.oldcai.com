给 Hexo 加上语义搜索：hexo-semantic-search-ai

oldcai
2026-01-15
Hexo
Hexo, SemanticSearch, 搜索, 语义搜索

静态博客很轻便，但搜索往往是短板：读者明明知道“要找什么”，却很难用关键词精准命中。对拥有 Hexo 博客的人来说，理想的搜索应该是“会理解意图的搜索”。

hexo-semantic-search-ai 就是为这个场景准备的：它把 SemanticSearch 语义搜索引擎接入 Hexo，让静态博客也能拥有“理解语义”的搜索体验，而且搜索引擎本身可以基于 Cloudflare Workers 的免费额度起步。

核心要点（站在 Hexo 博主的角度）

  1. 静态博客也能有真正可用的搜索
  • 不需要把站点改成动态服务
  • 不需要自建复杂后端
  • 只需部署一个语义搜索服务并接入插件
  1. 语义搜索比关键词更懂读者
  • 读者可以用自然语言描述问题
  • 搜索能理解上下文和意图，而不是死匹配
  1. 构建期同步，运行期几乎零负担
  • 文章在 hexo generate 后自动同步
  • 只同步变化内容（增量更新），维护成本低
  1. 相关文章更“像真的相关”
  • 基于语义相似度生成推荐
  • 更适合引导读者继续阅读
  1. 前端完全可控
  • 自带搜索框 helper 和 JS API
  • 你可以完全定制样式与渲染

适合谁？

  • 已经在用 Hexo，但发现“搜索体验不够好”的博主
  • 内容逐渐变多，想让读者更容易发现旧文
  • 想要搜索功能，但不想引入复杂后端或额外服务的人
