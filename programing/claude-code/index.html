Claude Code, Vibe Coding

Claude Code 可以做什么

所有的 Claude Code 所能做的都是基于一个基本的能力，就是对 Terminal 也就是终端的使用。你就想象一个 senior 程序员，他可以在控制台里面做的事情，基本上 Claude Code 都可以帮你办到。

快速理解

帮你快速的理解一个新的代码库。

和 Git 一起用的话，你可甚至可以让 Claude Code 把这一个星期代码的改变编成一个故事讲给你听。

做耗费时间的 Research

比如说你正在构思一个塔防游戏的实现方案。可以让Claude Code去搜索和对比几个框架，然后回头来告诉你，几个最好的方案。他会浏览和验证一些信息，然后呈现给你，你可以根据它来做出决定。

真正构建代码

用 Claude Code 可以帮你写好代码，然后提交 PR。你也不需要想在提交的消息里面写什么，他都会给你写好。

管理部署

用 Claude Code 可以帮你部署，你可以让它用调用云服务商的 API 去部署你的程序。

代码重构

因为 Claude Code 可以理解大量的代码，所以一些老的项目，一般人不敢去碰的代码，它也可以去阅读。
在一些旧的版本的 Java 或者是 PHP 程序升级的时候，它对团队来说有巨大的作用。
